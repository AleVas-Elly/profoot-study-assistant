# How to Run Profoot on Windows

1. Extract the downloaded folder into a normal location on your Computer (e.g., Desktop or Documents).
2. Double-click the file named `Start Chatbot (Windows).bat`.
3. The script will automatically check for Python and install required dependencies.
4. If Python is missing, follow the provided link and ensure you check the "Add Python to PATH" box during installation.
5. Once setup is complete, your web browser will automatically open Profoot.

### Note:
Do not close the terminal window while using the application; it runs the server in the background. Close the window when you are finished to stop the server.
